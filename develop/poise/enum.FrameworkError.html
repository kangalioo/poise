Expand description

Any error that can occur while the bot runs. Either thrown by user code (those variants will have an error field with your error type E in it), or originating from within the framework.

These errors are handled with the crate::FrameworkOptions::on_error callback

Expected a certain argument type at a certain position in the unstructured list of arguments, but found something else.

Most often the result of the bot not having registered the command in Discord, so Discord stores an outdated version of the command and its parameters.
